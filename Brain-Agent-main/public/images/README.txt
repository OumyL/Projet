Instructions pour ajouter les logos BRAIN :

L'application utilise deux logos différents pour le mode clair et le mode sombre :

1. Mode clair : brain-logo-l.png
2. Mode sombre : brain-logo-d.png

Veuillez ajouter ces deux fichiers dans ce répertoire (public/images/) :

Recommandations :
- Taille idéale : 400-600px de largeur pour une excellente visibilité
- Ratio largeur/hauteur recommandé : environ 3:1 pour un logo en bannière ou 1:1 pour un logo carré
- Format : PNG avec transparence
- Optimisez chaque logo pour son mode respectif :
  * Mode clair (brain-logo-l.png) : couleurs optimisées pour un fond clair
  * Mode sombre (brain-logo-d.png) : couleurs optimisées pour un fond sombre

Note : Les logos seront redimensionnés automatiquement pour s'adapter aux différentes zones de l'interface.
Si vous ne fournissez pas les deux logos, un texte "BRAIN" s'affichera à la place.
